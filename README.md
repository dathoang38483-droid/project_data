# Sales Data Analysis Project

This is a mini-project using Python and the Pandas library to read, inspect, and clean sales data from a CSV file.

## 📂 Project Structure

The project includes the following main files:

* `sales_data.csv`: The original dataset containing customer and purchase information.
* `task1.py`: The main Python script for data processing.
* `README.md`: Project documentation (this file).
